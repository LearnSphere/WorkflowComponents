kfKettle: A Knowledge Flow plugin that serves as a data source for data coming from the Kettle ETL tool.

URL:http://wiki.pentaho.com/display/DATAMINING/Using+the+Knowledge+Flow+Plugin
Author:Mark Hall <mhall{[at]}pentaho.com>
Maintainer:Mark Hall <mhall{[at]}pentaho.com>

A Knowledge Flow plugin that acts as a dummy data source for data coming from the Kettle ETL Tool. Is designed to be used as an entry point into a Knowledge Flow process by the Knowledge Flow Kettle plugin step. This Kettle component converts data rows from an upstream ETL process into Weka instances and then pumps them into the Knowledge Flow via this data source. This release includes an icon for the component (missing from the 1.0.0 release).
