JDBCDriversDummyPackage: Dummy package that provides a place to drop JDBC driver jar files so that they get loaded by the system.

Author:Mark Hall
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

Dummy package that provides a place to drop JDBC driver jar files so that they get loaded by the system. JDBC driver jar files may be placed in either ${WEKA_HOME}/packages/JDBCDriversDummyPackage or ${WEKA_HOME}/packages/JDBCDriversDummyPackage/lib

All available versions:
Latest
1.0.0
